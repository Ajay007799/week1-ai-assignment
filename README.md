# API & JSON Task

This project uses the free public Chuck Norris JSON API:
https://api.chucknorris.io/jokes/random

## Run
```bash
pip install requests
python main.py
```

## Files
- `main.py` — fetches live JSON, parses it, and prints formatted output
- `screenshots/` — contains sample screenshots of API response and execution output
- `output.txt` — created when the program runs
